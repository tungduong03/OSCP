DNS là 1 cơ sở dữ liệu phân tán, chịu trách nhiệm phân giải tên miền thành IP

`host <domain>`

Brute-force hostname

Tạo 1 danh sách hostname (list.txt)

`for ip in $(cat list.txt); do host $ip.domain.com; done`

Scan ip

`for ip in $(seq 200 254); do host 51.222.169.$ip; done | grep -v "not found"`

Kali hỗ trợ nhiều tool để DNS Enum: DNSRecon và DNSenum

### DNSRecon

`dnsrecon -d <domain> -t std` -> -d : domain, -t: kiểu scan (std: standard)

`dnsrecon -d <domain> -D /path/to/wordlist -t brt` -> -t brt (brute-force)

### DNSenum

`dnsenum <domain>`
